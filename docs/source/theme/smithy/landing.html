A framework for building machine learning enhanced media workflows
Get started

Focus on writing your workflows

Focus on writing code that creates your workflow instead of the resources or services needed to run the workflow. MIE automatically provisions an end-to-end workflow runtime and data interaction layer.

Familiar REST APIs

MIE provides familiar REST APIs for interacting with workflows and machine learning derived metadata. Skip the learning curve and get up and running quickly.

Deployed with one click via AWS Cloudformation

You can get started with MIE in just one click by launching the hosted AWS Cloudformation template on the project GitHub page.

Features

IAM Authentication

Built-in support for IAM Authentication. No extra services to learn for security.

Simple Workflow Schema

Reduced workflow complexity by utilizing a simplified workflow schema built on top of AWS StepFunctions.

Out-of-the-box Machine Learning

Pre-built machine learning workflow operators allow you to utilize many AWS AI/ML services immediately after deployment.

Full Testing Suite

Test your workflows with the provided testing suite.

Growing Developer Community

Learn and share amongst a growing developer community. Chat with the development team on Gitter.

Infinite Workflow Potential

Create custom workflow operators based on AWS Lambda to perform nearly any media task.
